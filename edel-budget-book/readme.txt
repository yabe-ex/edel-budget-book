投資やギャンブルの収支結果を記録できます。

== Description ==
収支を記録するとカレンダー上で日々の収支が表示され、収支一覧やチャートが作成されます。
日本円のみ対応しており、日本語圏のみの配布を想定しています。
ログインしたユーザーのみが利用でき、各ユーザーは公開/非公開を設定できます。
公開設定をすると、ログインユーザだけではなく非ログインユーザも収支結果を閲覧できます。

= 使い方 =
1. 設定画面で、項目を入力してください
　1行に1項目ずつ、「keyword:表示項目名,」の組み合わせを入力し、最終行のみカンマを入れないでください
2. 固定ページに、ショートコード[ebb]を入力すると収支表が設置されます
3. ログインしたユーザーは、カレンダーから日にちを選んでプラスボタンを押すと、収支内容を記録できます
4. URLの引数に「?member=ユーザーID」を指定すると、指定したユーザーIDのユーザーの収支結果が表示されます
5. 公開中/非公開中ボタンで非公開中に切り替えると、他のユーザーから閲覧できなくなります。

== Installation ==
1. このページからプラグインをダウンロードします
2. WordPressの管理画面から "プラグイン" -> "新規追加"
3. "プラグインのアップロード"を押し、ダウンロードした.zipファイルを選択します
4. "今すぐインストール"を押した後、"プラグインを有効化"を押します
